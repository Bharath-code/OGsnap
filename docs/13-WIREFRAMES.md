## Responsive Breakpoints

| Breakpoint | Width | Layout Changes |
|------------|-------|----------------|
| Mobile | < 640px | Single column, bottom nav, stacked cards |
| Tablet | 640-1024px | Two column, collapsible sidebar |
| Desktop | > 1024px | Full sidebar, three column stats |
| Wide | > 1440px | Centered content, max-width 1400px |

---

## Design Checkpoints

Before shipping any screen, verify:

- [ ] All interactive elements have visible focus states
- [ ] Color contrast meets WCAG AA (4.5:1 for text)
- [ ] All images have alt text
- [ ] Forms have associated labels
- [ ] Touch targets are minimum 44x44px on mobile
- [ ] Loading states don't exceed 3 seconds
- [ ] Error messages are clear and actionable
- [ ] Animations respect prefers-reduced-motion
- [ ] Dark mode is default (light mode optional)
- [ ] No layout shift on content load
