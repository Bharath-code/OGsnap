## 1. Dashboard Overview Philosophy

The dashboard is where users spend their time after converting. Good dashboard UX:
- Shows value immediately (stats, progress)
- Guides to next action (setup, upgrade)
- Celebrates milestones (progress, achievements)
- Reduces support burden (clear information)
